# Overview

MintQuotes is an open-source Bitcoin knowledge layer.

It transforms Bitcoin knowledge into structured, reusable data that developers, educators, communities, wallets, and applications can build upon.

What began as a simple Bitcoin quote project is evolving into open infrastructure for Bitcoin education.

Our mission is simple:

> **Help people understand Bitcoin—not just how to use it, but why it exists.**

---

# Why MintQuotes?

Bitcoin adoption is not only a technology challenge.

It is an understanding challenge.

Millions of people interact with Bitcoin without understanding concepts like:

- Money
- Inflation
- Scarcity
- Self-custody
- Privacy
- Decentralization
- Lightning
- Sovereignty

The Bitcoin ecosystem has built incredible financial tools.

MintQuotes focuses on the missing layer:

**Knowledge.**

# Architecture

MintQuotes is built around three connected layers.

## Knowledge Layer

The foundation.

Structured Bitcoin knowledge including:

- Quotes
- Concepts
- Authors
- Topics
- Categories
- Sources
- Learning resources

---

## Experience Layer

The MintQuotes web application.

Features include:

- Daily Bitcoin insights
- Search
- Topic exploration
- Author pages
- Knowledge pages
- Learning journeys

---

## Developer Layer

Reusable infrastructure for developers.

Planned capabilities include:

- Public API
- JavaScript SDK
- Widgets
- Open datasets
