# Overview

MintQuotes is an open-source Bitcoin knowledge infrastructure project.

It started as a minimalist Bitcoin quote application and is evolving into a foundation for Bitcoin education that developers, communities, educators, and applications can build on.

The mission is simple:

**Help people understand Bitcoin.**

Not only how to use Bitcoin.

But why Bitcoin exists.

---

# The Problem

Bitcoin adoption is not only a technology challenge.

It is an understanding challenge.

Millions of people interact with Bitcoin without understanding:

- Money
- Inflation
- Scarcity
- Self-custody
- Privacy
- Decentralization
- Lightning
- Sovereignty

The ecosystem has built powerful financial tools.

MintQuotes focuses on the missing layer:

> Knowledge.

# What MintQuotes Is Becoming

MintQuotes is designed as three connected layers.

---

# 1. Knowledge Layer

The foundation.

A structured collection of Bitcoin knowledge objects.

Each knowledge object contains:

- Concept
- Explanation
- Source
- Category
- Difficulty level
- Language
- Related resources


Example:

```json
{
  "id": "bitcoin-money-001",
  "topic": "Money",
  "level": "Beginner",
  "content": "Bitcoin introduces digital scarcity.",
  "languages": [
    "en"
  ],
  "sources": []
}
```

---

# 2. User Experience Layer

The MintQuotes application.

Current direction:

- Daily Bitcoin wisdom
- Educational cards
- Topic discovery
- Search
- Sharing
- Learning paths

The application is the first interface.

The knowledge layer is the foundation.

---

# 3. Developer Infrastructure Layer

The long-term goal is making Bitcoin education easy to integrate.

Future capabilities:

## API

Applications can request:

```
GET /quote/today

GET /topic/money

GET /learning/beginner
```

---

## SDK

Developers can integrate:

- Web components
- React components
- Mobile components
- Wallet integrations

---

## Widgets

Bitcoin education anywhere.

Example:

```html

topic="bitcoin">
</mintquotes-widget>
```
